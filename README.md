🏥 HEALTHCARE – Full Stack Healthcare Management System
<BR>
A full-stack web application designed to simplify healthcare service management by connecting patients, doctors, and administrators on a single platform.

🚀 Overview<BR>
HEALTHCARE is a modern web-based healthcare management system that enables:<BR>
🧑‍⚕️ Doctor profile management<BR>
📅 Appointment booking system<BR>
👤 Patient registration & authentication<BR>
📂 Medical record handling<BR>
🔐 Secure login & role-based access<BR>

The system is built using modern web technologies with separate frontend (client) and backend (server) architecture.
